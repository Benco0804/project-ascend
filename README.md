# project-ascend

Project Ascend is a mobile-first gamified habit-building app for healthy lifestyle.

The product helps people build healthier habits through daily quests, adaptive goals, identity-based progress, streaks, XP/levels, and AI coaching.

Ascend is not a calorie tracker.  
Ascend is not a traditional fitness app.  
Ascend is a behavior-change system for healthy living.

---

## Product Focus

Project Ascend focuses on helping users become healthier through repeated small actions.

Core areas:

- Nutrition habits
- Movement habits
- Training consistency
- Sleep and recovery
- Water intake
- Identity-based progress
- AI-guided habit adaptation

---

## Core Principles

1. Reward consistency, not perfection.
2. Reward actions, not outcomes.
3. Focus on healthy lifestyle habits, not only weight loss.
4. One small habit at a time.
5. The app should make users want to come back tomorrow.
6. AI should act as a coach, not a judge.
7. No feature enters MVP unless it improves retention or habit formation.

---

## MVP Direction

The first version will focus on:

- Simple onboarding
- Daily quest
- Habit check-in
- XP / streak / level
- Adaptive next mission
- Weekly AI review
- Basic progress screen

---

## Not in MVP

The first version will intentionally avoid:

- Calorie tracking
- Food database
- Barcode scanner
- Full workout builder
- Social feed
- Leaderboards
- Complex coach dashboard
- Wearable integrations
